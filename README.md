# QFM Agent — Releases

Signed APK releases for the QFM Agent companion app (Meta Quest 3/3S).

## Latest: v1.0.0

**Direct download:** [qfm-agent-1.0.0.apk](https://github.com/lcadhd/qfm-agent-releases/raw/main/releases/v1.0.0/qfm-agent-1.0.0.apk)

### Install via ADB
```bash
adb install qfm-agent-1.0.0.apk
```

### Deploy via HMS
Use this URL in Admin Center > Devices > Apps > + New app:
```
https://github.com/lcadhd/qfm-agent-releases/raw/main/releases/v1.0.0/qfm-agent-1.0.0.apk
```

### Requirements
- Meta Quest 3 or Quest 3S
- Horizon OS v57+ (Android API 32+)
- Developer mode enabled
